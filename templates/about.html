Youtube Comment App

This project was created using the flask microframework in python. It uses the Youtube Data Api v3 to extract the information from youtube. The user will be prompted to enter a video ID and also a number of top comments to show. If given these parameters and submitted, the system will sort all the comments of that video and return the top comments of the video. Note: the more comments on a video, the longer the process takes to sort. Only press submit one time or reset the parameters by clicking home on the nav bar.
